# PresenceIX & Folio Studio — Digital Suite & Founder Portfolio

A premium, interactive digital suite built completely from scratch using semantic **HTML5**, **Vanilla CSS**, and **JavaScript**. This repository showcases high-performance front-end engineering, modular interface design, fluid layouts, custom-curated design systems, and responsive layouts tailored to high-conversion business needs.

## 🚀 Projects Included

1. **PresenceIX (Modern Cyberpunk Real Estate AI Agency Landing Page)** (`index.html` & `tbl/index.html`)
   - **Theme:** Vibrant, high-contrast dark mode with neon accents (`#00ffaa`, `#00c8ff`).
   - **Key Features:** Animated particulate background, glassmorphic dashboards, live lead simulations with interactive status counters, SVG flow diagrams, marquee-based list tracks, and CSS smooth transitions.
2. **Folio Studio (Premium Editorial Web Design Studio Landing Page)** (`tbl/folio-studio.html`)
   - **Theme:** Warm, organic editorial palette ("paper and ink" feel) with deep rust accents (`#c4521a`).
   - **Key Features:** Custom cursor-ring following interactions, custom SVG noise filters for rich paper texture, 3D perspective mockups using purely CSS transitions, interactive service grids, step-by-step process boards, and a responsive contact system integrated with WhatsApp.
3. **Kachi Ezibe's Founder Portfolio (Minimalist Editorial Personal Site)** (`tbl/portfolio.html`)
   - **Theme:** Elegant dark editorial look featuring serif typography and gold accents (`#c9a84c`).
   - **Key Features:** Full-viewport photography integration, high-end hover-reveal micro-animations on timeline components, 3-dimensional preview cards, and email client integration.

## 🛠️ Tech Stack & Engineering Highlights

- **Pure Frontend Core:** Built purely with raw **HTML5**, **CSS3**, and **ES6+ JavaScript**. No bloated JavaScript frameworks, static site builders, or heavy utility libraries like Tailwind. This guarantees sub-second load times and ensures pixel-perfect cross-browser layouts.
- **Custom CSS Design Tokens:** Leverages CSS custom properties (`:root` variables) for modular tokens, enabling unified typography, responsive scales, transitions, shadows, and smooth light-to-dark theme toggling on the fly.
- **Micro-Interactions & Animation:** Incorporates highly engaging cursor trail rings, particle flow engines, hover-to-reveal panels, perspective card-rotation, and smooth page anchors to create a premium, tactile feel.
- **Design Typography:** Features hand-curated fonts from Google Fonts: *Cabinet Grotesk* (bold geometric sans-serif), *Instrument Serif* (elegant editorial serif), *Cormorant Garamond* (classical serif), *Outfit*, and *Bebas Neue*.
